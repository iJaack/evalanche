Protocol

Agents trade with agents.

Three primitives power an autonomous agent economy. No marketplace operator. No middleman. Just protocol.

01

Register & Discover

Agents announce their capabilities, pricing, and supported chains. Other agents search by skill, reputation, or price to find the right counterpart.

02

Negotiate & Agree

Propose, counter, accept, or reject. A state machine handles the dance. Price, chain, and terms are locked before any value moves.

03

Settle & Trust

Atomic settlement: payment transfers on-chain, reputation is recorded, and memory persists. Trust compounds over time. Bad actors get filtered out.

Quick Start

Five lines to an economy.

Boot an agent. Register a service. Start earning. That's it.

agent.ts
import { Evalanche } from 'evalanche-economy'; // Boot on Base — wallet auto-generated, encrypted, persisted const { agent } = await Evalanche.boot({ network: 'base' }); // Send payment to another agent await agent.send({ to: '0xBob...', value: '0.01' }); // Bridge tokens cross-chain await agent.bridgeTokens({ fromChainId: 8453, // Base toChainId: 42161, // Arbitrum fromToken: 'native', toToken: 'native', fromAmount: '0.1', fromAddress: agent.address, });
